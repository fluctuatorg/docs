## Required Values (For Cross Border Payouts)

As a prerequisite to creating a Cross Border Payout, you are required to send us the ultimate sender and recipient’s information. Learn the required information for the region to start sending payouts:

### Sender

| Field Parameter | Type | Description |
| --- | --- | --- |
| `individual_detail.first_name` | `INDIVIDUAL` | First name of sender |
| `individual_detail.last_name` | `INDIVIDUAL` | Last name of sender |
| `business_detail.business_name` | `BUSINESS` | Full registered name of sender |
| `business_detail.business_domicile` | `BUSINESS` | Full address of sender |
| `addresses.country` | Any | Country address of sender |
| `addresses.province_state` | `BUSINESS` | Province/State address of sender |
| `addresses.city` | `BUSINESS` | City address of sender |
| `addresses.street_line1` | `BUSINESS` | Full address of sender |

### Recipient

| Field Parameter | Type | Description |
| --- | --- | --- |
| `individual_detail.first_name` | `INDIVIDUAL` | First name of recipient |
| `individual_detail.last_name` | `INDIVIDUAL` | Last name of recipient |
| `business_detail.business_name` | `BUSINESS` | Full registered name of recipient |
| `identity_accounts.type` | Any | Account type of the recipient’s account |
| `identity_accounts.company` | Any | Channel code of the recipient’s account |
| `identity_accounts.country` | Any | Issuing country of the recipient’s account |
| `identity_accounts.properties.account_number` | Any | Account number of the recipient’s account |
| `identity_accounts.properties.account_holder_name` | Any | Account holder name of the recipient’s account |
