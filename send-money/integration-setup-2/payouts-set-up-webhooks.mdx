Learn how you can be notified of any status changes for your payouts in real-time. For the best payout experience, we highly recommend you to subscribe to our webhooks.

## Set up your Webhook URL

Important: You need `Developer` access to set up

Follow the following instructions to start setting up your webhook URL:

1. Log in to your Xendit Dashboard
2. Open [Webhook Settings](https://dashboard.xendit.co/settings/developers#webhooks)
3. Configure your URL on each webhook event you want to subscribe
4. Click Test and Save to receive test webhooks

For more information on the different webhook events that you can subscribe, refer to your corresponding payout product integration page.
