---

## Features & Requirements

|  |  |
| --- | --- |
| Channel code | `SCB_MOBILE_BANKING` |
| Currency | THB |
| Minimum amount | 20 |
| Maximum amount | 700,000 |
| User approval flow | REDIRECT |
| Reusable payment code | ❌ |
| Custom payment code | ❌ |
| Display merchant name | Xendit |
| Display user name | ❌ |
| Set expiry | ❌ |
| Settlement Time | Instant |
| Refund | ❌ |
| Partial refund | ❌ |
| Multiple partial refund | ❌ |
| Refund validity | ❌ |
| Compatible integration | Payment API, Payment Link |

2. Redirect end customer to SCB Mobile Banking app
3. End customers confirm the amount
4. End customer click confirm the payment
