Indodana PayLater is a digital credit service that allows users to purchase goods or services on credit at various merchants partnered with Indodana Finance.

---

## Features

|  |  |
| --- | --- |
| Channel code | `INDODANA` |
| Currency | IDR |
| Minimum amount | 10,000 |
| Maximum amount | 25,000,000 |
| User approval flow | REDIRECT |
| Save | ❌ |

| Auth & capture | ❌ |
| Partial capture | ❌ |
| Multiple partial capture | ❌ |
| Payment request expiry | 24 hours |
| Payment token validity | ❌ |
| Settlement time | T+2 working days |
| Refund | ✅ |
| Partial refund | ✅ |
| Multiple partial refund | ❌ |

## Payment flow

```mermaid
sequenceDiagram
    participant Customer
    participant Merchant as Your app/server
    participant Xendit
    participant Indodana

    Customer->>Merchant: Start checkout
    Merchant->>Xendit: Create payment request (INDODANA)
    Xendit-->>Merchant: Return redirect/action URL
    Merchant-->>Customer: Redirect to Indodana approval page
    Customer->>Indodana: Complete approval
    Indodana-->>Xendit: Return payment result
    Xendit-->>Merchant: Send payment webhook
    Merchant-->>Customer: Show final payment status
```
