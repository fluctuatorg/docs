> Supported only for ID and PH businesses.

Wix is a cloud-based web development platform that lets users create HTML5 websites and mobile sites using drag-and-drop tools. Its eCommerce system allows you to sell a wide range of products with easy setup.

Follow this guideline to start integrating Xendit with your Wix platform.

## Requirements

Before integrating Xendit with Wix, ensure the following:

1. Xendit Registration:

   - Register with Xendit to obtain your test and live API keys.
2. Wix Account:

   - Sign up for Wix and subscribe to the Business Plan or eCommerce Plan package.

## Installation

Here are the steps you need to do to integrate Xendit and your Wix store:

<iframe src="https://www.youtube.com/embed/cCGLfNHxocg" title="YouTube video player" className="w-full rounded-xl" height="315"></iframe>

1. Go to *Setting > Accept Payments* in your Wix dashboard and choose "Online Payments by Xendit" in More Payment Options, click Connect
2. Input Xendit API key to the Secret Key field in your Wix dashboard Xendit setting

   1. To process test transaction, please input Secret Key from Xendit Dashboard in Test environment
   2. To process real transaction, please input Secret Key from Xendit Dashboard in Live environment
3. Click Connect
