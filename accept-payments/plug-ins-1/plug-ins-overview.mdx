With third-party platforms like Shopify, WooCommerce, and others, you can easily create an online store without needing to code. This allows you to focus on what really matters: your customers, products, and services.

Xendit provides ready-made plugins for these platforms, enabling you to effortlessly accept payments online. With fast and automated payment processing, you can reliably manage payments from your customers without any hassle.

## Set up Plug-ins

If you're developing your website using plugins and want to accept payments via Xendit, your integration with the plugins must meet our requirements. Follow these steps to get started:

1. Create a Xendit account  
   Sign up for a Xendit account to begin your integration. See the guidelines [here](/get-started/create-account#how-to-sign-up).
2. Create Xendit API key

   Create each environment’s Xendit api key to be used for the plug-in installation. See the guidelines [here](/account-security/api-keys#how-to-generate-api-keys). For plug-in usage, the API key only needs to have money-in access.
3. Understand the payment lifecycle  
   Review the [payment lifecycle](/accept-payments/integration-guide/payment-link/payment-links-api-overview#payment-link-lifecycle) to understand how payments flow through the system.
4. Follow plugin requirements & installation guidelines  
   Refer to the individual documentation for each plugin you want to implement, where you'll find specific requirements and installation instructions.
